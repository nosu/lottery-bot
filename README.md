# ちゅうせんくん for LINE

## 概要
改行区切りで人名を送ると、その中から指定の人数をランダムで選んでくれるLINE Botです。

## 使い方
1. `$ git clone https://github.com/nosu/lottery-bot.git`
2. `$ cp .env.sample .env && vi .env`
3. `$ docker-compose build`
4. `$ docker-compose up`
